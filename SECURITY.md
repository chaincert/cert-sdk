# Security Policy

## Supported Versions
CertID handles critical hardware-anchored identity infrastructure. We actively support and patch the current major release:

| Version | Supported          |
| ------- | ------------------ |
| v0.1.19  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a potential security vulnerability in the CertID SDK, the Stylus WASM contracts, or our infrastructure, please **do not** publicly disclose it by opening a GitHub issue. 

Public disclosure puts our institutional partners and users at risk. 

Instead, please report it privately via email to: **admin@c3rt.org**

**Response SLA:**
* We will acknowledge receipt of your vulnerability report within 48 hours.
* We will strive to send you regular updates about our progress in verifying and patching the issue.
* If the vulnerability is accepted, we will coordinate a secure patch and a synchronized public disclosure timeline with you.
